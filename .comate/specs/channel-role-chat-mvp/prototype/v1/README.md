# prototype v1 · channel-role-chat-mvp

## 目的

这个 HTML 是 Morrow 频道/角色/聊天 MVP 的可交互原型，用来验证基础体验，而不是最终实现代码。

## 可点击路径

1. 左侧 `频道 +` 新建频道，填写频道名，选择绑定文件夹，选择初始 AI 成员。
2. 左侧 `AI 队友 +` 新建角色，走完模板、身份、加入频道三步。
3. 在频道输入框里点击 `@设计师` 或手动输入 `@`，选择角色并发送。
4. 点击频道成员或消息头像，打开角色详情。
5. 在角色详情里切换 `Chat` / `Settings`，测试私聊和基础设置。

## MVP 边界

- 做频道、文件夹绑定、角色、成员关系、频道聊天、角色详情。
- 不做任务抢单、自动 handoff、Memory、Skills、Activity 等高级模块。
- 目标是让“创建频道 → 创建角色 → 加入频道 → @聊天 → 查看角色”这条基础链路顺畅。
