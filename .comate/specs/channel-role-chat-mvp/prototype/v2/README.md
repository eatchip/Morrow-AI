# Morrow Channel Role Chat MVP v2

这个版本基于 v1 的反馈重新收窄为“本地个人工作空间”：

- 左侧只保留个人对话、群聊频道、AI 队友。
- 新建群聊通过“选择本地文件夹...”表达系统文件夹选择器，不再让用户从固定列表里选。
- 新建角色包含名称、模型、简介、指示、加入群聊；模型必选 `Claude Code` 或 `Codex`。
- 「指示」是角色 prompt 的主要定义，用户可在角色 Settings 中随时查看和修改。
- 如果当前没有群聊，仍可先创建角色，之后从群聊里添加。
- 频道成员栏默认不常驻，可从顶部按钮展开或收起。
- 角色详情只保留 `Chat` 和 `Settings` 两个必要页签。
