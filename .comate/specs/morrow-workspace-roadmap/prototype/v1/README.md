# prototype v1 · morrow-workspace-roadmap

## 目的

用一个静态 HTML 把 Morrow 从当前形态到“AI 同事工作空间”的阶段性前端样子讲清楚。

## 打开方式

双击 `index.html` 即可打开，不需要构建、不需要联网。

## 本轮关注点

1. 每个阶段的前端信息架构是否能一眼看懂
2. Stage 1 是否足够小，适合作为下一轮正式 SDD
3. “AI 同事”是否被放在 Task / Run / Timeline / Approval 这些工作对象里，而不是变成多聊天窗口
4. 你是否认同先做 Workspace Kernel，再做多 Agent

## 非目标

- 不代表最终视觉稿
- 不改 Morrow 当前生产代码
- 不一次性承诺所有阶段都实现
